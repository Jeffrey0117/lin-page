# **📌 問題 1：品牌介紹頁（圖 1）

→ 左圖 + 右文字的排版在這個寬度下比例失衡造成破版**

### **❗ 具體問題點：**

### ① 左側照片比例太固定（可能用 `width:50%` 或固定 px）

導致在「介於 768–1024px」這種尺寸時，左圖吃太多空間 → 文字區變得太窄。

### ② 右側文字的「寬度限制太小 + padding 過厚」

推測你右半部是用：

* `width: 50%`
* 外面還有 `padding: 40px`
* 欄寬變成 50% - padding → 最後只能剩下類似 300px（超難閱讀）

### ③ 中文換行 + 字型太細 → 中段文字看起來被壓縮

在某些 RWD 斷點中文字本來就會變高，現在右側又太窄 → 變成一長條文字柱。

### ④ 整體沒進到「平板版」

看起來你只有：

* 桌機版（2 欄）
* 手機版（1 欄）

但 **缺少「平板版 1 欄置中」**
→ 就發生你現在看到的半破版狀況。

---

# **📌 該怎麼修（很簡單）：**

### **✔ 方案 A：在 1024px 下改成 1 欄（最普遍）**

平板不要雙欄，就不會擠。

```
@media (max-width: 1024px) {
  .about-section {
    display: block;
  }
  .about-image,
  .about-text {
    width: 100%;
  }
}
```

### **✔ 方案 B：保留雙欄，但設定 min-width**

例如：

```
.about-text {
  min-width: 420px;
}
```

防止文字區太窄。

# **📌 問題 2：頂部導覽列（圖 2）

→ 字太多 + 沒換行邏輯 → 撐到超出 / 擠壓 Logo**

### **❗ 具體問題點：**

### ① 菜單項目全部以「inline-block + 空白」呈現

導致所有字被硬塞在同一行，沒有進入「摺行模式」。

### ② Navbar 寬度固定（看起來高度也固定）

一旦文字超出可用空間，就會：

* 擠壓 Logo 字距
* 看起來「左右不對稱」
* 項目太密集沒有留白

### ③ 沒有替換成「漢堡選單」的 RWD breakpoints

你的這條很顯然應該在：

* 1200px 保持一行
* 1024px 改字距、縮 Logo
* 768px 直接改成 hamburger menu

但你現在看起來連 **1024px** 都沒 RWD。

---

# **📌 該怎麼修：**

### **✔ 在 1024px 下開始縮排 / 減字距**

```
@media (max-width: 1024px) {
  .nav-menu {
    font-size: 14px;
    gap: 16px;
  }
  .logo {
    letter-spacing: normal;
  }
}
```

### **✔ 在 768px 以下直接改 Hamburguer menu**

你只有這樣 navbar 才會真的穩。

---

# **📌 總結（你要的重點）**

### **圖 1（品牌介紹）破版原因：**

* 左圖佔比太高
* 右文字欄太窄（只有桌機 / 手機版，沒有中間版）
* 建議 1024px 以下直接改成單欄

### **圖 2（Navbar）破版原因：**

* 字太多沒有 wrap 邏輯
* 1024px 沒任何 RWD
* 768px 必須 Hamburger menu
